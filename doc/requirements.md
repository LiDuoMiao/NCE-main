## 需求详情

### 需求001：单词点击翻译

**功能描述：**
在双语歌词区，点击任意英文单词，弹出翻译词典风格的弹窗。

**弹窗内容：**
1. 单词本身
2. 音标
3. 发音：英语发音 + 美语发音（可点击播放）
4. 词性（noun, verb, adj. 等）
5. 释义（中文解释）
6. 3个例句

**弹窗交互：**
- 弹窗显示在屏幕正中央，与单词位置无关
- 点击关闭按钮（×）可关闭弹窗
- 点击弹窗外部区域可关闭弹窗

**歌词区域交互调整：**
- 每行歌词右侧增加一个独立的播放按钮（喇叭图标）
- 点击播放按钮 → 播放该句音频
- 点击单词 → 弹出翻译弹窗
- 两者独立，不冲突

**技术实现：**
- 调用 Google Translate API 获取翻译结果
- 单词用 `<span>` 包裹，绑定点击事件
- 弹窗宽度约 320px

**样式要求：**
- 与页面现有样式保持一致（圆角、阴影等）

## 需求详情

### 需求002：PDF教材查看功能

**功能描述：**
在页面右侧添加 PDF 教材预览区域，用户可查看对应课本的 PDF 内容。

**布局调整：**
```
┌─────────────────────────────────────────────────┐
│  Header (全宽) - 不变                            │
├─────────────────────────────────────────────────┤
│  Book info + Course selection (全宽) - 不变       │
├───────────────────────────┬─────────────────────┤
│  Player controls          │                     │
│  (宽度50%)               │   PDF Viewer        │
├───────────────────────────┤   (宽度50%)        │
│  Lyrics/Content area     │                     │
│  (宽度50%)               │                     │
└───────────────────────────┴─────────────────────┘
```

**技术实现：**
- 使用 pdf.js 库渲染 PDF
- PDF 文件放在 `pdf/` 目录作为静态资源
- 用户选择不同课本时，切换加载对应的 PDF 文件

**交互说明：**
- PDF 区域可滚动，用户自行翻页查找内容
- 不做页码与课程的自动对应，由用户自行翻阅

### 需求003：学习笔记功能

**功能描述：**
在页面右侧 PDF 区域顶部添加两个可切换的标签：`课本` 和 `学习笔记`。点击标签可切换显示内容，学习笔记与左侧课程选择联动。

**布局调整：**
```
┌─────────────────────────────────────────────────┐
│  Header                                          │
├─────────────────────────────────────────────────┤
│  Book info + Course selection                    │
├───────────────────────────┬─────────────────────┤
│  Player controls           │  [课本] [学习笔记]   │
│  音频播放控制区            ├─────────────────────┤
│                           │                     │
│  ─────────────────────    │   PDF Viewer        │
│                           │        或            │
│  双语歌词/对话显示区       │   学习笔记内容       │
│  (左侧)                   │   (根据标签切换)     │
│                           │                     │
└───────────────────────────┴─────────────────────┘
```

**标签切换逻辑：**
- 默认显示"课本"（即 PDF 内容）
- 点击"学习笔记" → 右侧切换为笔记内容
- 左侧选不同的课 → 右侧笔记内容也跟着变（按下标对应）
- 点击"课本" → 切回 PDF 显示

**笔记内容（AI 生成）：**
- **核心词汇**：最多 15 个（不设下限，AI 根据内容自适应）
- **重点短语**：最多 10 个（不设下限，AI 根据内容自适应）
- **语法讲解**：1-3 个重点语法点
- **句型仿写**：2-4 个重点句型

**课程与笔记对应关系：**
- 使用下标/索引对应（而非标题匹配）
- 左侧课程列表[0] → 笔记文件 001-002.json
- 左侧课程列表[1] → 笔记文件 003-004.json
- 以此类推

**笔记内单词交互：**
- 笔记中的单词可点击，支持查词典功能（与左侧歌词区一致）

**技术实现：**
- 笔记生成脚本：`scripts/generate_notes.py`
- 调用 MiniMax API 生成笔记（JSON 格式）
- 输出目录：`notes/NCE{1,2,3,4}/`
- 前端复用车已有的查词典功能

### 需求004：第一册问答音频功能

**功能描述：**
新概念英语第一册的每课包含两部分内容：对话（奇数课）和问答（偶数课）。当前系统只能播放对话部分，需要增加"对话/问答"切换功能。

**布局调整：**
```
┌─────────────────────────────────────────────────────────────┐
│  《NCE》1st Level    [对话|问答]    选择课程[001&002]        │
├─────────────────────────────────────────────────────────────┤
│  [◀] [▶/⏸] [▶]  播放速度    [中英]    (音频控制栏)          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  左侧对话/歌词显示区                                           │
│  (对话模式：显示双语歌词)                                      │
│  (问答模式：隐藏此区域)                                       │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

**按钮样式：**
- 参考现有的 `tab-btn` 样式
- 两个按钮并排：[对话] [问答]
- 默认"对话"选中（背景 accent-1，白色文字）
- "问答"未选中时显示为普通样式
- 悬停时背景变 accent-3

**切换逻辑：**

| 模式 | 音频加载 | 箭头按钮 | 中英切换 | 下方列表 |
|------|---------|---------|---------|---------|
| 对话 | 对话音频（现有） | 显示 | 显示 | 显示 |
| 问答 | 问答音频（qa_tape/index.json） | 隐藏 | 隐藏 | 隐藏 |

**数据文件：**
- 文件路径：`qa_tape/index.json`
- 结构：key 为课程 index（0-71），value 为 Q&A 音频 URL
- 示例：`{"0": "https://file.enfamily.cn/nce/book1/NCE1002.mp3", ...}`

**问答音频 URL 规则：**
- 课程 index 0（001-002）→ lesson 2 → `NCE1002.mp3`
- 课程 index 1（003-004）→ lesson 4 → `NCE1004.mp3`
- 依此类推：lesson_number = (index + 1) × 2
- URL 格式：`https://file.enfamily.cn/nce/book1/NCE1{lesson_number}.mp3`

**技术实现：**
- 切换按钮只在 NCE1 时显示，其他册隐藏
- 切换到"问答"时：
  - 音频 src 换成 index.json 对应的 URL
  - 隐藏 `.nav-btn.prev`、`.nav-btn.next`、`.lang-toggle`
  - 隐藏 `.lyrics-container`
- 切换回"对话"时恢复所有元素

**交互说明：**
- 切换按钮位于页面顶部（书名右侧）
- 切换模式后，音频自动加载对应内容
- 问答模式下的播放控制（播放/暂停/速度）与对话模式一致
