## 项目概述

新概念英语·全四册在线点读系统 - 一个纯静态的英语学习Web应用。

- **在线体验**: http://nce.ichochy.com
- **技术栈**: HTML + CSS + JavaScript（纯静态实现）
- **功能**: 美音课文朗读、单句点读、中英对照
